spawn-process

spawn-process   name function &rest args => process[Function]

Purpose
Spawn a new process.

Package   :portable-threads

Module   :portable-threads

Arguments

name     A string
function     A function
args     Arguments to the function

Returns
The new process.

Errors
Multiprocessing is not supported on the Common Lisp implementation.

Example 

> (spawn-process "Sleepy" #'sleep 60)
#<process Sleepy>
