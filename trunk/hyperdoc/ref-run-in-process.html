run-in-process

run-in-process   process function &rest args[Function]

Purpose
Force process to apply function to args

Package   :portable-threads

Module   :portable-threads

Arguments

process     A process
function     A function
args     Arguments to the function

Errors
Multiprocessing is not supported on the Common Lisp implementation.

Description
The behavior of calling run-in-process with a process that is hibernating is undefined. Some Common Lisp implementations may not call function until the process is awakened.

See also
    spawn-process

Example 

  (run-in-process process 
                  #'(lambda (result) (throw ':exit result)) 
                  result)
