close-network-streamer

close-network-streamer   network-streamer[Function]

Purpose
Close a network streamer (connection to an external streamer node).

Package   :gbbopen

Module   :network-streaming

Arguments and values

network-streamer     A network streamer

Errors
Threads (multiprocessing) is not supported on the Common Lisp implementation.

See also
    define-streamer-node
    find-streamer-node
    close-network-streamer
    open-network-streamer

Example
Close the network streamer *streamer*: 

  (close-network-streamer *streamer*)
