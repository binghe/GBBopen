signal-event

signal-event   event-class &rest initargs[Function]

Purpose
Signal an event

Package   :gbbopen

Module   :gbbopen-core

Arguments and values

event-class     An event class or a non-nil, non-keyword symbol that names an event class
initargs     An initialization argument list

Description
The following table lists the initialization arguments that are required for specific event metaclasses:

      Event metaclass Required initargs
non-instance-event-class None
instance-event-class :instance unit-instance
space-instance-event-class      :instance unit-instance
:space-instance space-instance
nonlink-slot-event-class :instance unit-instance
:slot effective-nonlink-slot-definition
link-slot-event-class :instance unit-instance
:slot effective-link-definition

See also
    define-event-class
    with-events-disabled
    with-events-enabled

Example 

  (signal-event 'my-event :my-event-arg1 3)
