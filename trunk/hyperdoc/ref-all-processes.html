all-processes

all-processes   <no arguments> => list-of-processes[Function]

Purpose
Return a list of all processes.

Package   :portable-threads

Module   :portable-threads

Returns
A list of objects representing the processes.

Description
On Common Lisp implementations without multiprocessing, nil is returned.

See also
    current-process
    processp
    spawn-process

Example 

> (all-processes)
(#<process Listener 1>)
