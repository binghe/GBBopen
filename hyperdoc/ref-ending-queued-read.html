ending-queued-read

ending-queued-read   tag[Generic Function]

Purpose
Perform completion activities associated with reading a queued-streamer block.

Method signatures

ending-queued-read  (tag t)

Package   :gbbopen

Module   :streaming

Arguments and values

tag     An object

Description
Ending-queued-read is called when the end of a queued-streamer block is read; it should not be called directly. It will always be called with the same (but not necessarily eq) tag value as the corresponding beginning-queued-read. The matching ending-queued-read will not be called if reading of the queued-streamer block is not completed due to error (such as a premature ending of the input stream).

See also
    beginning-queued-read
    with-queued-streaming
    write-streamer-queue

Example
Write a method that prints a notice when an empty queued-streamer block (tagged with the tag :empty) has been read: 

  (defmethod ending-queued-read ((tag (eql ':empty)))
    (format t "~&;; An empty queued-streamer block was read.~%"))
