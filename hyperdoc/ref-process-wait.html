process-wait

process-wait   whostate predicate &rest args[Function]

Purpose
Suspend a process until applying predicate to args returns true.

Package   :portable-threads

Module   :portable-threads

Arguments

whostate     A string
predicate     A function that returns ageneralized boolean
args     Arguments to the predicate function

See also
    hibernate-process
    process-wait-with-timeout
    process-whostate

Example 

  (process-wait "Waiting for Godot" #'gate-open-p godot-gate)

Notes
The whostate value is ignored by SBCL.
