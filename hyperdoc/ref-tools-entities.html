GBBopen Tools

The GBBopen Tools module, :gbbopen-tools, contains useful Common Lisp additions and utilities. Convenient (shorthand operators for declared fixnum, short-float, single-float, double-float, and long-float numeric operators are also provided by the :gbbopen-tools module. Date and time parsing and formatting entities and efficient offset-universal-time functions are also included in GBBopen Tools.

Uniform interfaces to implementation-specific thread (multiprocessing) capabilities are provided by the :portable-threads module, polling functions for Common Lisp implementations that do not provide thread capabilities are provided by the :polling-functions module, uniform socket interfaces are provided by the :portable-sockets module, and uniform interfaces to the operating system are provided by the :os-interface module.

The GBBopen Tools module imports commonly used CLOS and MOP symbols into the :gbbopen-tools package and then exports them, making it easy to use CLOS and MOP entities by simply using the :gbbopen-tools package. Load-time warning messages are displayed if the Common Lisp implementation does not have one of these symbols in its CLOS or MOP implementation.

Several entities in the :gbbopen-tools module that are extended by additional methods in the :gbbopen-core module are documented in the GBBopen Core section of this Reference. These and related entities that are documented in the GBBopen Core section are: *print-object-for-sending*, *save/send-references-only*, initialize-saved/sent-instance, make-duplicate-instance, omitted-slots-for-saving/sending, print-object-for-saving/sending, print-slot-for-saving/sending, with-reading-saved/sent-objects-block, and with-saving/sending-block.
