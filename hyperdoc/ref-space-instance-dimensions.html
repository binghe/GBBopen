space-instance-dimensions

space-instance-dimensions   space-instance => dimension-list[Generic Function]

Purpose
Return the dimensions of a space instance.

Method signatures

space-instance-dimensions   (space-instance standard-space-instance) => dimension-list

Package   :gbbopen

Module   :gbbopen-core

Arguments

space-instance     A space instance

Returns
A list of dimension name, dimension type pairs.

See also
    make-space-instance
    unit-class-dimensions

Example
Return the dimensions of the (bb hyps) space instance: 

> (space-instance-dimensions (find-space-instance-by-path '(bb hyps))
((x :ordered) (y :ordered) (belief :ordered) (classification :enumerated))
