list-length-1-p

list-length-1-p   list => boolean[Function]

Purpose
Fast length = 1 test of a list.

Package   :gbbopen-tools

Module   :gbbopen-tools

Arguments and values

list     A proper list or a dotted list
boolean     A generalized boolean

Returns
True if list has length 1; nil otherwise.

See also
    list-length>
    list-length>1
    list-length>2
    list-length-2-p

Examples 

  > (list-length-1-p '(a))
  t
  > (list-length-1-p '(a b))
  nil
  > (list-length-1-p nil)
  nil
  > (list-length-1-p '(a . b))
  nil
  >
