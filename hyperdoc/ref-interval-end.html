interval-end

interval-end   interval => end-value[Function]

Purpose
Obtain the end value of an interval.

Setf syntax

(setf (interval-end   interval) end-value)

Package   :gbbopen

Module   :gbbopen

Arguments

interval     An interval

Returns
The end value of the interval.

See also
    expand-interval
    interval-end
    shift-interval

Examples 

> (interval-end '(1 2))
2
> (interval-end '(1 . 2))
2
> (interval-end #(1  2))
2
