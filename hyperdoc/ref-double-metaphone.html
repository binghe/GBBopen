double-metaphone

double-metaphone   string => primary-index, secondary-index[Function]

Purpose
Compute the primary and secondary Double Metaphone phonetic-code strings of string.

Package   :gbbopen-tools

Module   :double-metaphone

Arguments and values

string     A string
primary-index     A string
secondary-index     A string or nil

Returns
Two values: the primary and secondary phonetic-code strings. The second value is nil, if there is no secondary code for string.

Example 

  > (double-metaphone "testing")
  "TSTN"
  nil
  > (double-metaphone "Smith")
  "SM0"
  "XMT"
  > (double-metaphone "Schmidt")
  "XMT"
  "SMT"
  >
