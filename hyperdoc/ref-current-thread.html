current-thread

current-thread   <no arguments> => thread[Function]

Purpose
Return the object representing the current thread.

Package   :portable-threads

Module   :portable-threads

Returns
The object representing the current thread.

See also
    all-threads
    spawn-thread

Example 

> (current-thread)
#<thread Listener 1>

Note
On Common Lisp implementations without threads, :threads-not-available is returned.
