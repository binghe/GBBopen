shuffle-list

shuffle-list   list => shuffled-list[Function]

Purpose
Return a copy of a list, with the elements in random order.

Package   :gbbopen-tools

Module   :gbbopen-tools

Arguments

list     A proper list

Returns
The shuffled copy of list.

Examples 

> (shuffle-list '(a b c d))
(b a c d)
> (shuffle-list '(a b c d))
(c a d b)
