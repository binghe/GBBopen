symbol-value-in-process

symbol-value-in-process   symbol process => object, boundp[Function]

Purpose
Return the value of symbol in a process.

Package   :portable-threads

Module   :portable-threads

Arguments

symbol     A symbol
process     A process

Returns
Two values:

  • the value of symbol in process or nil if no value is bound
  • t if symbol is specially or globally bound in process; otherwise nil

Errors
Multiprocessing is not supported on the Common Lisp implementation.

Description
The global symbol value is returned as the first value if no process-local value is bound.

The behavior of calling symbol-value-in-process with a process that is hibernating is undefined. Some Common Lisp implementations may not return until the process is awakened.

See also
    spawn-process

Examples 

> (symbol-value-in-process '*x* process)
33
t
> (symbol-value-in-process 'pi process)
3.141592653589793d0
t
> (symbol-value-in-process '*unbound* process)
nil
nil
