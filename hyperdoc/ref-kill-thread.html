kill-thread

kill-thread   thread[Function]

Purpose
Kill a thread.

Package   :portable-threads

Module   :portable-threads

Arguments

thread     A thread

Errors
Threads (multiprocessing) is not supported on the Common Lisp implementation.

See also
    spawn-thread
    thread-alive-p

Example 

  (kill-thread thread)
